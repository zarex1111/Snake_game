Файлы игры

    Snake - основной цикл и файлы

        class Snake

            run_game - механизм игрового цикла

            _check_events - часть основного цикла, анализирующая нажатия кнопок клавиатуры

            _update_screen - рисует объекты на экране

            _update_bodies - обновляет "тело" змейки

            _check_head_walls и _check_head_body - проверяет столкновения с объектами

            _check_play/speed_button - обновляет кнопки

            new_game - сбрасывает файлы игры

        class Apple

            Пассивный объект, судьба которого быть съеденным

        class Body

            Класс сегмента тела змеи, знающий положение и направление следующего сегмента

            _move - обновляет координаты 

            _change - меняет изображение согласно изменениям в направлении

        class Button 

            Кнопка любого назначения

        class Head 

            Первый и управляемый сегмент змейки

            Меняет изображение и направление согласно действиям пользователя

        class Score - текстовое окно с результатом

        class Settings - необходимые настройки

        class Speed_Button - кнопка изменения скорости

        class Stats - записывает данные в json-файл

    data и images - папки с изображениями и json-файлом
